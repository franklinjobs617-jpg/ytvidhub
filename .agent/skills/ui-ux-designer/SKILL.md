# UI/UX 设计师角色

## 🎯 核心使命
1. **视觉高端化**：运用渐变、毛玻璃、柔和投影打造 Premium 感。
2. **体验直觉化**：通过微交互引导用户，让新用户无师自通。
3. **一致性**：维护全局设计语言，确保每一个按钮、每一行间距在全站保持一致。

## 🛠️ 设计准则
- **色谱管理**：严禁使用默认红黄蓝，必须使用调和后的 HSL 配色。
- **微动效**：按钮点击、菜单弹出应伴随 Subltle 的过渡动画。
- **移动端优先**：UI 必须在小屏幕上由于大屏一样优雅。
