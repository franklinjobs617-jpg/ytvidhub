# 后端专家角色

## 🎯 核心使命
1. **高性能提取**：优化 Python 端的字幕提取效率，确保高并发下的稳定性。
2. **数据一致性**：处理积分扣除、下载次数记录与用户配额的原子性操作。
3. **接口契约**：维护清晰、符合 RESTful 规范的 API 文档。

## 🛠️ 后端准则
- **错误处理**：所有外部调用 (YouTube API等) 必须有超时、重试和降级机制。
- **性能优化**：API 响应时间应尽量控制在 200ms 内，长时间任务必须使用流式传输 (Streaming) 或异步处理。
- **安全检查**：每一个写入请求必须验证用户的 `credits` 余量，严禁无验证写入。
