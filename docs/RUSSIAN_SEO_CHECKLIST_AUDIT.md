## 7. 用户友好性检查 ⭐ 重点

### ⚠️ 发现的问题

#### 问题1: Hero Subtitle 过长且重复
**当前翻译**：
```
Вставьте ссылку и скачайте субтитры с Ютуба за несколько секунд. Поддержка видео, плейлистов и каналов в форматах SRT, VTT, TXT. Как скачать субтитры с YouTube - просто вставьте ссылку.
```

**问题**：
- 最后一句"просто вставьте ссылку"与开头"Вставьте ссылку"重复
- 整体过长，不够简洁

**建议优化**：
```
Вставьте ссылку и скачайте субтитры с Ютуба за секунды. Поддержка видео, плейлистов и каналов в форматах SRT, VTT, TXT. Как скачать субтитры с YouTube? Просто вставьте URL.
```

#### 问题2: 部分翻译过于直译
**featuresGrid.items.llm.title**: "Питайте вашу LLM"

**问题**：
- "Питайте"（Feed）在俄语中用于LLM不够自然
- 更自然的表达应该是"Обучайте"（Train）或"Используйте для"（Use for）

**建议优化**：
```
"Обучайте вашу LLM" 或 "Данные для вашей LLM"
```

#### 问题3: CTA按钮文案可以更友好
**hero.cta.primary**: "Скачать субтитры с YouTube"

**当前状态**: 功能性强，但不够吸引人

**建议优化**（可选）：
```
"Скачать субтитры бесплатно" (更强调免费)
```
