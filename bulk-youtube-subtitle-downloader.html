Technical Q&A

Is there a limit to how many URLs I can paste?
For free users, there is a daily limit. For Pro members, there are absolutely no limits on the number of URLs you can process in a single job.
How are files named in the ZIP?
We automatically name each file after the original YouTube video title. No more "videoplayback.srt".
Can I download as TXT?
Yes! Before downloading, you can select SRT, VTT, or TXT format. The entire batch will be converted to your choice.
