# Workspace 优化计划

> 目标：让用户进来就觉得"这比那些垃圾下载站好用多了"

## 现状分析

### 已有功能（基础扎实）
- 字幕点击跳转视频时间戳 ✅
- 字幕自动滚动跟随播放位置 ✅
- AI Summary + Study Cards ✅
- 智能模式 / 列表模式切换 ✅
- 搜索字幕 ✅
- 复制全文 ✅

### 当前视觉现状（对比 YouMind）

| 项目 | 当前 | YouMind | 问题 |
|------|------|---------|------|
| 背景色 | `bg-white` 纯白 | 深色 `#0d0d0d` | 显得像普通网页，缺乏沉浸感 |
| 当前行高亮 | `bg-violet-50` 极浅 | 左侧竖线 + 微亮背景 | 几乎看不出来 |
| 时间戳颜色 | 活跃 `violet-700`，非活跃 `blue-600` | 统一灰色 | 两种颜色混用，视觉不统一 |
| 非活跃行文字 | `text-slate-700` 正常亮度 | 透明度降低 | 当前行无法突出 |
| 行间距 | `space-y-3` | 更宽松 | 偏紧，阅读感差 |
| 面板分隔 | border | 极细线 | 边界感太强 |

### 核心问题
1. **视觉质感不足** - 白色背景 + 混乱的颜色系统，缺乏专业感
2. **字幕体验不够丝滑** - 点击跳转有延迟感，高亮不够明显
3. **AI Summary 加载体验差** - 等待时用户不知道发生了什么
4. **布局拥挤** - 三栏（视频+字幕+AI）在小屏幕上体验差
5. **缺少"哇"时刻** - 没有让用户眼前一亮的交互细节

## 优化 Todolist（按优先级）

### P0 - 视觉质感（最快见效）

- [ ] **字幕区背景** - `bg-white` → `bg-slate-50`，减少刺眼感
- [ ] **当前行高亮重做**
  - 左侧加 3px 紫色竖线（`border-l-[3px] border-violet-500`）
  - 背景保持 `bg-violet-50`
  - 非当前行文字透明度降到 60%（`opacity-60`）
  - 效果：当前行一眼就能找到
- [ ] **时间戳颜色统一** - 全部改为 `text-violet-500`，去掉 `blue-600`
- [ ] **行间距** - `space-y-3` → `space-y-4`，呼吸感更好
- [ ] **面板分隔线** - `border-slate-100/50` 改为 `border-slate-100`，更细腻

### P1 - 交互体验（核心差异化）

- [ ] **字幕高亮动画**
  - 点击字幕跳转后，该行有短暂闪烁（`animate-pulse` 一次）
  - 让用户感知到"跳转成功了"

- [ ] **字幕自动滚动优化**
  - 当前行保持在字幕区域垂直居中（`scrollIntoView({ block: 'center' })`）
  - 用户手动滚动时暂停自动跟随，3秒后恢复（加 `isUserScrolling` flag）

- [ ] **搜索关键词高亮**
  - 搜索结果中匹配词用黄色标记（`<mark>` 标签）
  - 显示匹配数量 "3 / 47"

- [ ] **字幕行 hover 复制**
  - 鼠标悬浮时右侧出现复制图标
  - 复制格式：`[00:01:23] 字幕内容`

### P2 - 细节打磨

- [ ] **键盘快捷键** - `Ctrl+F` 聚焦搜索框
- [ ] **AI Summary 打字机光标** - 流式生成时末尾显示闪烁光标
- [ ] **移动端 Tab 布局** - 视频 / 字幕 / AI Summary 三个 Tab

### P1 - 差异化功能（让用户觉得"专业"）

- [x] **字幕搜索体验优化**
  - 搜索结果高亮关键词（黄色标记）
  - 显示匹配数量 "3 / 47 matches"
  - 支持上下翻页浏览搜索结果
  - 当前搜索结果特殊高亮显示

- [x] **一键复制当前段落**
  - 鼠标悬浮在字幕行时，右侧出现复制图标
  - 复制格式：`[00:01:23] 字幕内容`
  - 增强的复制反馈动画

- [x] **AI Summary 流式显示优化**
  - 打字机效果逐字显示（30ms间隔）
  - 生成中显示实时字数统计
  - 摘要完成后有完成动画
  - 流式打字光标效果

### P2 - 细节打磨（提升整体质感）

- [x] **空状态优化**
  - 首次进入 workspace 无视频时，显示引导动画
  - 提示"粘贴 YouTube 链接开始分析"

- [x] **键盘快捷键**
  - 空格键：播放/暂停
  - ←/→：快退/快进 5 秒
  - Ctrl+F：聚焦搜索框

- [x] **字幕导出增强**
  - 导出时包含视频标题和 URL
  - 支持带时间戳的 TXT 格式

- [x] **移动端布局**
  - Tab 切换：视频 / 字幕 / AI Summary
  - 底部固定 Tab Bar
